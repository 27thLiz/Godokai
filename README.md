# Godokai
Color scheme for the Godot script editor, inspired by Monokai.

Replace the color definitions in `editor_settings.xml` (located in ~/.godot/ on linux) with:
```xml
		<color name="text_editor/background_color"> 0.105882, 0.113725, 0.117647, 1 </color>
		<color name="text_editor/caret_color"> 0.666667, 0.666667, 0.666667, 1 </color>
		<color name="text_editor/line_number_color"> 0.666667, 0.666667, 0.666667, 0.4 </color>
		<color name="text_editor/text_color"> 0.636719, 0.636719, 0.611847, 1 </color>
		<color name="text_editor/text_selected_color"> 0.609375, 0.597473, 0.597473, 1 </color>
		<color name="text_editor/keyword_color"> 0.972549, 0.14902, 0.439216, 1 </color>
		<color name="text_editor/base_type_color"> 0.976471, 0.14902, 0.447059, 1 </color>
		<color name="text_editor/engine_type_color"> 0.976471, 0.14902, 0.447059, 1 </color>
		<color name="text_editor/function_color"> 0.4, 0.635294, 0.807843, 1 </color>
		<color name="text_editor/member_variable_color"> 0.670588, 0.862745, 0.286275, 1 </color>
		<color name="text_editor/comment_color"> 0.501961, 0.501961, 0.501961, 1 </color>
		<color name="text_editor/string_color"> 0.901961, 0.858824, 0.454902, 1 </color>
		<color name="text_editor/number_color"> 0.682353, 0.505882, 1, 1 </color>
		<color name="text_editor/symbol_color"> 0.729412, 0.87451, 1, 1 </color>
		<color name="text_editor/selection_color"> 0.188235, 0.215686, 0.215686, 1 </color>
		<color name="text_editor/brace_mismatch_color"> 1, 0.2, 0.2, 1 </color>
		<color name="text_editor/current_line_color"> 0.3, 0.5, 0.8, 0.15 </color>
		<color name="text_editor/word_highlighted_color"> 0.8, 0.9, 0.9, 0.15 </color>
```
